ABOUT US

This system can tell whether a user trying to log in to the system is a legitimate user or not. When a user tries to log in, the camera will capture the user's image. If the user and the record match, a pop-up message will appear confirming that the user is valid, and the user will then have access to the system and its data. And if the user is unidentified then the pop up message will appear confirming that user is invalid. The camera will capture the invalid user and will save the image in the desktop. We have added a unique feature in this system which prevents the invalid user from repeatedly logging in. The system will then block that user.

  • Features includes various dockers operations like list docker containers, stop docker containers, start docker containers etc.
